Для хранения данных используется СУБД MS SQL Server.
Подключение устанавливается к локальному серверу.
В файле EffM.sql скрипт для создания тестовой базы.

В базе 4 таблицы:
- Таблица районов (4 района)
- Таблица заказов
- Таблица логов (указывается район время формирования, время заказа и количество заказов)
- Таблица результатов (указывается район и время заказа)

Тестовые данные заполняются за 28.10.2024.

Также результат записывается в текстовый файл, который формируется рядом с exe файлом.

Для формирования результатов фильтрации нужно запустить TestEffM.exe и выбрать необходимые параметры, затем нажать на кнопку "Сформировать".
